Variables and Data Types in JavaScript


In JavaScript, variables are used to store data values. There are several different data types that can be stored in a variable, including:

  • Number: represents numeric values, both integers and floating-point numbers.
  • String: represents a sequence of characters enclosed in quotes, either single ('') or double ("").
  • Boolean: represents a logical value that can be either true or false.
  • Null: represents an intentional absence of any object value.
  • Undefined: represents a variable that has been declared but has not been assigned a value.
  • Object: represents a collection of key-value pairs or properties, where each property can be of any data type.

Variables are declared using the var, let, or const keyword, depending on how you want to use them.

Here's an example of declaring and initializing a variable: 

        
            let myNumber = 42;

In this case, myNumber is a variable of the number data type, and it has been initialized with the value 42.

For example, to declare a variable called message with the string value "Hello, World!", you can use the following code: 

        
            const message = "Hello, World!";

Here, const is the keyword that declares the variable, message is the name of the variable, and "Hello, World!" is the value assigned to it.

While var has been traditionally used in JavaScript, its use has become somewhat outdated with the introduction of let and const.

The main issue with var is that it has function scope, not block scope. This means that if you declare a variable with var inside a function, it will be accessible throughout the entire function, regardless of where it was declared. This can sometimes lead to unintended consequences and make it harder to reason about your code.

In contrast, let and const have block scope, meaning that they are only accessible within the block they were declared in. This can make your code easier to read and understand, and help prevent errors.

It's important to note that in JavaScript, variables are dynamically typed, which means that you don't need to specify the data type of the variable when you declare it.

Instead, the data type is determined at runtime based on the value assigned to the variable.

Here are some examples of variable declarations and data types:

    

    var age = 25; // Number data type
    var name = "John"; // String data type
    var isMarried = false; // Boolean data type
    var car = null; // Null data type
    var address; // Undefined data type
    var person = {name: "John", age: 25}; // Object data type

In JavaScript, you can also use the typeof operator to check the data type of a variable: 

    
    console.log(typeof age); // Output: "number"
    console.log(typeof name); // Output: "string"
    console.log(typeof isMarried); // Output: "boolean"
    console.log(typeof car); // Output: "object"
    console.log(typeof address); // Output: "undefined"
    console.log(typeof person); // Output: "object"

Strict Mode

Strict mode is a way to introduce a secure coding standard in JavaScript. It helps to catch coding mistakes that might lead to errors or unexpected behavior. Strict mode is a feature introduced in ECMAScript 5 (ES5).

Strict mode enforces a set of rules and restrictions on your JavaScript code. These rules are intended to prevent common coding mistakes, and to make it easier to write secure, optimized code. Here are some of the restrictions imposed by strict mode:

  • Implicit declaration of variables is not allowed. All variables must be declared before use.
  • Assigning a value to a read-only property or a property that does not exist in an object results in an error.
  • Deleting a variable or a function is not allowed.
  • Function parameters must have unique names.
  • Using the 'with' statement is not allowed.
  • Octal numeric literals are not allowed.
  • Use of the 'eval' function is restricted.

Strict mode can be enabled in two ways:

  • By adding the string "use strict" at the beginning of a script or a function.
  • By enabling it for the entire script file by putting "use strict" at the beginning of the file.

Here is an example of how to enable strict mode in a function:

        function myFunction() {
        "use strict";
        // Function code goes here
        }

Here is an example of how to enable strict mode for an entire script file:

        "use strict";
        // JavaScript code goes here

It's important to note that strict mode is not backward compatible with older versions of JavaScript. This means that if you're upgrading an older codebase to ES5, you'll need to check for any code that might break under strict mode and make the necessary changes.



Lambda Functions in JavaScript

Lambda functions, also known as anonymous functions, are a feature in JavaScript that allows you to define and call a function without assigning it a name. They're typically used for short, simple functions that don't need to be reused in multiple places.

Defining Lambda Functions

In JavaScript, you can define a lambda function using the () => {} syntax. The parentheses are used to pass in arguments, if any, and the curly braces are used to define the body of the function. Here's an example: 

      
        const multiply = (a, b) => {
          return a * b;
        };
        console.log(multiply(3, 4)); // Output: 12

In this example, we're defining a lambda function called multiply that takes in two arguments, a and b, and returns their product. We're then calling the function with the arguments 3 and 4 and logging the result to the console.

Uses of Lambda Functions

Lambda functions are commonly used in JavaScript for tasks such as event handling, array manipulation, and asynchronous programming. For example, here's how you can use a lambda function to sort an array of objects by a specific property: 

      
        const people = [
          { name: 'Alice', age: 25 },
          { name: 'Bob', age: 30 },
          { name: 'Charlie', age: 20 }
        ];
        const sortByAge = (a, b) => {
          return a.age - b.age;
        };
        const sortedPeople = people.sort(sortByAge);
        console.log(sortedPeople);
        // Output:
        // [
        //   { name: 'Charlie', age: 20 },
        //   { name: 'Alice', age: 25 },
        //   { name: 'Bob', age: 30 }
        // ]

In this example, we're defining a lambda function called sortByAge that takes in two objects and returns the difference between their ages. We're then using the sort() method on the people array to sort it by age using the sortByAge lambda function. We're finally logging the sorted array to the console.



Iteration in JavaScript

Loops are a fundamental concept in programming, and they are used to execute a block of code repeatedly. In JavaScript, there are several types of loops:

  • for loop
  • while loop
  • forEach()
  • filter()
  • map()

For loop

The for loop is used to execute a block of code a specific number of times. 

      
        for(let i = 0; i < 5; i++) {
          console.log(i);
        }

This code will have the same result as the previous.

While loop

The while loop is used to execute a block of code while a condition is true. It consists of only one part:

      
        let i = 0;
        while(i < 5) {
          console.log(i);
          i++;
        }

This code will print the numbers 0 to 4 in the console.

forEach()

The forEach() loop is similar to the for loop, but the method loops through the elements of an array/collection.

Here's an example:

      
        let letters = [a, b, c, d];

        letters.forEach(letter =>{
          console.log(`This is letter ${letter}`)
          }

Map()

The map() method is a built-in function in JavaScript that does not modify the original array, but instead returns a new array with the transformed elements. The map() method takes in a single argument, which will be called on each element of the original array. 

      
        const numbers = [1, 2, 3, 4, 5];
        const doubledNumbers = numbers.map(function(num) {
          return num * 2;
        });
        console.log(doubledNumbers); // Output: [2, 4, 6, 8, 10]

Filter()

The filter() method takes in a single argument, which is a callback function that will be called on each element of the original array. The callback function should return a boolean value that indicates whether the element should be included in the new array or not. 

      
        const numbers = [1, 2, 3, 4, 5];
        const evenNumbers = numbers.filter(function(num) {
          return num % 2 === 0;
        });
        console.log(evenNumbers); // Output: [2, 4]

Chaining Methods

One of the benefits this looping is that it can be chained, using filter(), map(), and sort(), to perform more complex data transformations. Here's an example: 

      
        const people = [
          { name: 'Alice', age: 25 },
          { name: 'Bob', age: 30 },
          { name: 'Charlie', age: 20 }
        ];
        const sortedNames = people
          .filter(function(person) {
            return person.age > 20;
          })
          .map(function(person) {
            return person.name;
          })
          .sort();
        console.log(sortedNames); // Output: ["Alice", "Bob"]


Date and Time in JavaScript


JavaScript provides a built-in object called Date, which allows developers to work with dates and times in their applications.

Creating a Date Object

To create a new Date object, you can use the new Date() constructor.

const currentDate = new Date();

This will create a new Date object representing the current date and time.

Working with Dates and Times

The Date object provides several methods for working with dates and times.


        // Get the current date and time
        const currentDate = new Date();
        console.log(currentDate);

        // Get the current year
        const currentYear = new Date().getFullYear();
        console.log(currentYear);

        // Get the current month
        const currentMonth = new Date().getMonth();
        console.log(currentMonth);

        // Get the current day of the month (1-31)
        const currentDay = new Date().getDate();
        console.log(currentDay);

        // Get the current hour (0-23)
        const currentHour = new Date().getHours();
        console.log(currentHour);

        // Get the current minute (0-59)
        const currentMinute = new Date().getMinutes();
        console.log(currentMinute);

        // Get the current second (0-59)
        const currentSecond = new Date().getSeconds();
        console.log(currentSecond);

        // Get the current millisecond (0-999)
        const currentMillisecond = new Date().getMilliseconds();
        console.log(currentMillisecond);

        // Set the date to January 1, 2022
        const newDate = new Date(2022, 0, 1);
        console.log(newDate);

        // Set the time to 3:30 PM
        newDate.setHours(15);
        newDate.setMinutes(30);
        console.log(newDate);

Formatting Dates and Times

JavaScript also provides methods for formatting dates and times into strings. 

        
            // Format the date as YYYY-MM-DD
            const currentDate = new Date();
            const formattedDate = currentDate.toISOString().slice(0,10);
            console.log(formattedDate);
